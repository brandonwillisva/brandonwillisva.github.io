🛠️ About This Project

This dashboard is built with HTML, CSS, and JavaScript — the three core languages of the web. Right now it uses simulated stock data (generated with math to look realistic).

Next step to make it real: Sign up for a free API key at alphavantage.co, then replace the generatePrices() function with a real fetch() call to their API. That's how professional FinTech apps work — they request live data from a server.

Ideas to level this up: Add a portfolio tracker, a stock comparison view, or a news feed using a free headlines API. Each addition is a new thing to learn and show off!
